# basic-traffic-monitoring-prototype
A basic traffic monitoring and adaptive signal simulation prototype using Python and  OpenCV. 

### Tasks:
• Load a traffic intersection video  
• Detect and count vehicles (use YOLO based models for detection and SORT/DeepSORT for tracking) 
• Estimate traffic density using vehicle count  
• Simulate adaptive traffic signal timing based on density (Python + OpenCV or SUMO) 
• Display congestion warning when vehicle count exceeds threshold  
• Compare fixed-time vs adaptive green signal allocation conceptually
